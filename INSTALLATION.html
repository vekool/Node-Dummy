Node Dummy - Installation Instructions

Prerequisites

Make sure you have Node.js installed on your system.

  • Download from: https://nodejs.org
  • Recommended: LTS version (Long Term Support)

Installation Steps

1. Extract the ZIP file

Unzip the project folder to your desired location.

2. Open Terminal/Command Prompt

Navigate to the project folder:

cd path/to/Node Dummy

3. Install Dependencies

Run the following command to install all required packages:

npm install

This will install:

  • express
  • body-parser
  • jsonwebtoken

4. Start the Server

Run the server using:

npm start

The server will start on http://localhost:3000

Available API Endpoints

  • POST /login - Get a random username and JWT token
  • POST /register - Mock registration
  • GET /profile - Get user profile (requires JWT)
  • POST /logout - Logout endpoint

For detailed API documentation, see api-documentation.html

Troubleshooting

  • Port already in use: If port 3000 is busy, stop other applications or modify the PORT value in server.js
  • npm not found: Make sure Node.js is properly installed and added to your system PATH
  • Permission errors: Try running your terminal as administrator (Windows) or use sudo (Mac/Linux)
